1. 评论总数

SELECT COUNT(*) 
FROM reviews
WHERE 
	products_id = ? AND
	status = 1

2. 评论星级

  • 最高5颗星
  • 不足1颗星部分：小于0.25部分，忽略；大于等于0.25并且小于0.75部分，取半颗星，大于0.75则取一颗星

2.1 星级计算

SELECT SUM(price_rating + quality_rating + usefullness_rating + easyuse_rating + service_rating) / COUNT(*) / 5 AS average_rating, COUNT(*) review_total 
FROM reviews
WHERE 
	status = 1
GROUP BY products_id;
