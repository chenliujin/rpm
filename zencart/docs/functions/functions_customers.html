zen_get_customer_right

$_COOKIE['right'] = 2

可以浏览大客户产品，可以使用分包

  • customers_other_info.use_package_manager = 1
  • 或者 $_SESSION ['bigger_customer'] == 1
  • 或者 SUM(orders.orders_total) > $50 && MIN(orders.date_purchased)距今超过 30 天

$_COOKIE['right'] = 1

可以浏览大客户产品，不可以分包

  • 购买过一次以上，但是金额和时间不满足要求的用户

$_COOKIE['right'] = 0

不可以浏览大客户产品，不可以分包

  • 一次也没买过的用户
