配置

$ cd /opt/solr/server/webapps/
$ unzip solr.war -d solr

$ cd /opt/tomcat/webapps/	
$ ln -s /opt/solr/server/webapps/solr solr

$ cp /opt/solr/server/lib/ext/* /opt/solr/server/webapps/solr/WEB-INF/lib/

# 指定 solr/home
$ vim /opt/tomcat/webapps/solr/WEB-INF/web.xml


$ vim /opt/tomcat/conf/server.xml

启动tomcat服务器

$ /opt/tomcat/bin/startup.sh

关闭防火墙

$ systemctl stop firewalld

Add Core

$ cd /opt/solr/server/solr/	
$ mkdir product
$ cp -R /opt/solr/server/solr/configsets/sample_techproducts_configs/conf ./product/
