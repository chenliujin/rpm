确定Hadoop版本

1.2.1-1

确定支持Hadoop 1.2.1-1版本的Hbase

hbase-0.98.8-hadoop1-bin.tar.gz

安装HBase

$ tar -xzvf hbase-0.98.8-hadoop1-bin.tar.gz	
$ mv hbase-0.98.8-hadoop1 /opt/
$ cd /opt/
$ ln -s hbase-0.98.8-hadoop1 hbase

安装HBase(此为错误的HBase版本，此版本仅支持Hadoop 2.x)

$ tar -xzvf hbase-1.0.0-bin.tar.gz
$ mv hbase-1.0.0 /opt/
$ cd /opt/
$ ln -s hbase-1.0.0 hbase
